## Dataset Description

The dataset used for flood prediction contains various weather and rainfall-related parameters to train the Machine Learning model.

### Input Features

The dataset includes:

- Temperature
- Humidity
- Cloud Cover
- Annual Rainfall
- January–February Rainfall
- March–May Rainfall
- June–September Rainfall
- October–December Rainfall
- Average June Rainfall
- Subdivision Rainfall

### Target Variable

- Flood Label

The **Flood Label** represents the output class used by the Machine Learning model to predict whether flood conditions are likely to occur.
