To identify the most suitable algorithm for flood prediction, four classification models were implemented and evaluated.

1.Decision Tree
The Decision Tree model classifies data by creating decision rules based on feature values. It is easy to understand and provides fast predictions. The model achieved an accuracy of 96.55% on the testing dataset.

2.Random Forest
The Random Forest algorithm combines multiple decision trees to improve prediction accuracy and reduce overfitting. It achieved an accuracy of 96.55% on the testing dataset.

3.K-Nearest Neighbors (KNN)
The KNN algorithm predicts the class of a new sample based on the majority class of its nearest neighbours. In this project, the KNN model achieved an accuracy of 89.66%.

4.Gradient Boosting
The Gradient Boosting model builds multiple weak learners sequentially, with each new model correcting the errors of the previous one. It achieved an accuracy of 96.55% and was selected as the final deployment model due to its strong predictive performance and good generalization capability.
