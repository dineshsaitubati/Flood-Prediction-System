## Proposed Solution

To overcome the limitations of existing flood prediction methods, a Machine Learning-based web application named **Rising Waters** was developed.

The system provides an automated flood prediction approach by:

- Collecting important weather-related parameters.
- Processing and analyzing the collected data using a trained Machine Learning model.
- Predicting whether flood conditions are likely to occur.
- Displaying prediction results through a user-friendly Flask-based web application.

The proposed system helps provide faster predictions, supports early warning capabilities, and assists in improving disaster preparedness and response.
