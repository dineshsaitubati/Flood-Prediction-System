## Installation Steps

Follow the steps below to set up and run the **Rising Waters Flood Prediction System** locally.

### Step 1: Install Python

Install **Python 3.x** on your system and ensure it is added to the system environment variables.

### Step 2: Clone the Repository

Download or clone the project repository from GitHub:

```bash
git clone <repository-url>
```

### Step 3: Install Required Dependencies

Navigate to the project directory and install the required libraries using:

```bash
pip install -r requirements.txt
```

### Step 4: Run the Flask Application

Start the application using:

```bash
python app.py
```

### Step 5: Access the Application

Open the generated localhost URL in your web browser to access the flood prediction application.
