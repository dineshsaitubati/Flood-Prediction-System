## Project Timeline

The development process of the **Rising Waters Flood Prediction System** was completed through the following phases:

| Week | Activity |
|------|----------|
| Week 1 | Literature Survey and Problem Identification |
| Week 2 | Dataset Collection and Requirement Analysis |
| Week 3 | Data Cleaning and Exploratory Data Analysis (EDA) |
| Week 4 | Data Preprocessing, Feature Scaling, and Model Development |
| Week 5 | Model Evaluation, Comparison, and Selection |
| Week 6 | Flask Application Development |
| Week 7 | Application Testing and Validation |
| Week 8 | Documentation, GitHub Upload, and Final Demonstration |
