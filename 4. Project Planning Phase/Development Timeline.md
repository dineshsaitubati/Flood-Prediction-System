## Project Timeline

The development process of the **Rising Waters Flood Prediction System** was completed within 4 weeks:

| Week | Activity |
|------|----------|
| Week 1 | Literature Survey, Problem Identification, Dataset Collection, and Requirement Analysis |
| Week 2 | Data Cleaning, Exploratory Data Analysis (EDA), Data Preprocessing, and Feature Scaling |
| Week 3 | Machine Learning Model Development, Evaluation, Model Selection, and Flask Application Development |
| Week 4 | Application Testing, Validation, Documentation, GitHub Upload, and Final Demonstration |
