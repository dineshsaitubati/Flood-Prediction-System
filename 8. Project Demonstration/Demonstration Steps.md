## Application Usage

Follow the steps below to use the Flask application:

1. Launch the Flask application.

```bash
python app.py
```

2. Open the Home page in the browser.

3. Navigate to the **Prediction** page.

4. Enter the required weather parameters.

5. Click the **Predict** button.

6. View the prediction result displayed by the application.

7. Repeat the process with different input values to verify both **Flood** and **No Flood** scenarios.
