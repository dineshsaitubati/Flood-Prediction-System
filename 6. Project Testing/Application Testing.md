## Application Testing and Validation

The Flask web application was tested to verify the functionality and reliability of the flood prediction system.

The testing process included:

- Checking page navigation between different application screens.
- Validating user input through the prediction form.
- Verifying the integration between the Flask application and the trained Machine Learning model.
- Testing prediction generation using different input values.
- Confirming proper display of prediction results.

The application successfully generated flood predictions and displayed the appropriate result pages based on the user-provided weather parameters.
