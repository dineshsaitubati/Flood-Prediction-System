## Model Performance Evaluation

The final **Gradient Boosting Model** achieved an accuracy of **95.65%** on the testing dataset.

The model performance was evaluated using the following evaluation metrics:

- **Confusion Matrix:** Used to analyze the number of correct and incorrect predictions made by the model.
- **Classification Report:** Used to evaluate the model based on:
  - Precision
  - Recall
  - F1-score

The evaluation results demonstrate that the Gradient Boosting model provides reliable flood prediction performance and is suitable for deployment in the Flask-based web application.
